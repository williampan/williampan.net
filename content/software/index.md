I have written two input schema for the [Rime input method engine](https://rime.im/): one for typing Standard Mandarin in pinyin, and one for typing Taiwanese in peh-oe-ji. These can be downloaded from their Github repositories: <span class="small-caps">[pinyin](https://github.com/williampan/rime-lumen-pinyin)</span>, <span class="small-caps">[peh-oe-ji](https://github.com/williampan/rime-taigi-poj)</span>.
